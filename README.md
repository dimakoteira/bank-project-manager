# 🏦 Bank Management System (C++)

A comprehensive console-based banking application developed in **C++**. This project demonstrates core programming concepts, including data structures, file handling, and system logic.

 🚀 Features
- **Client Management:** Add, Update, Find, and Delete client records.
- **Transactions:** Deposit and Withdraw money with balance validation.
- **Data Persistence:** All data is saved and retrieved from text files (`bank.txt`) using custom separators.
- **Security:** Account number verification to prevent duplicate entries.
- **User Interface:** Professional console menus with formatted tables using `<iomanip>`.

 🛠️ Technologies Used
- **Language:** C++
- **Data Storage:** File Streams (`fstream`)
- **Data Structures:** Vectors and Structs
- **Formatting:** `iomanip` for tabular data display

 📑 How it Works
The system loads client data into a `vector` of structs upon startup. It uses a custom string-splitting algorithm to parse data from the text file.
